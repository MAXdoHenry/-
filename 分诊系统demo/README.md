# 分诊系统 Demo

这是一个基于 **Python Flask + SQLite + Bootstrap 5 CDN** 的医联体分级会诊/分诊 Demo。

项目目标是演示：

- 医生登录
- 发起疑难病例提问
- 同级、上级、跨级医院请教
- 附件上传
- 会诊回复
- 回复通知
- 按可见医院等级控制查看权限

本项目是教学和演示用 Demo，不包含真实医疗数据。

## 1. 技术栈

- 后端：Python Flask
- 数据库：SQLite
- 前端：HTML + Bootstrap 5 CDN
- 附件：保存到本地 `uploads/` 文件夹

## 2. 目录结构

```text
app.py
```

```text

```

这些都是运行产物，不建议上传到 GitHub。

## 3. 如何运行

进入项目目录：

```bash
cd <项目目录>
```

安装依赖：

```bash
pip install flask
```

启动系统：

```bash
python app.py
```

启动成功后，终端会看到类似输出：

```text
Running on http://127.0.0.1:5000
```

本机浏览器访问：

```text
http://127.0.0.1:5000
```

如果要让同一局域网内的其他电脑或手机访问，请使用部署电脑的局域网 IP：

```text
http://<部署电脑IP>:5000
```

例如：

```text
http://192.168.x.x:5000
```

## 5. 功能说明

### 登录

访问系统后会先进入登录页。

未登录用户访问其他页面时，会自动跳回登录页。

### 首页

登录后进入首页，展示：

- 当前医生信息
- 我的提问数量
- 待回复数量
- 已回复数量
- 最新提问列表

### 发起提问

医生可以填写：

- 标题
- 病情描述
- 请教医院
- 附件
- 哪些等级医院可以查看此提问

请教医院支持同级、上级、跨级选择。

### 附件上传

提问和回复都支持附件。

当前支持：

```text
png、jpg、jpeg、gif、webp、pdf、mp4、mov、avi、webm
```

图片和视频会在详情页预览，其他附件提供下载。

### 我的提问

只显示当前登录医生自己发起的提问。

### 会诊提问列表

显示当前医生有权限查看的其他提问。

可见规则：

1. 自己发起的提问永远可见。
2. 其他提问是否可见，由提问人选择的“可见医院等级”决定。
3. 被指定为“请教医院”的医生可以查看并回复。
4. 只有被请教医院的医生可以回复，其他可见医院只能查看。

### 提问详情

详情页展示：

- 提问标题
- 提问人
- 所属医院
- 科室
- 提问时间
- 病情描述
- 附件
- 可见医院等级
- 回复内容

### 回复通知

当自己的提问被回复后，页面顶部会显示：

```text
您的提问已被回答
```

## 6. 如何验证 Demo

下面是一组推荐验证流程。

### 第一步：验证系统能启动

执行：

```bash
python app.py
```

浏览器访问：

```text
http://127.0.0.1:5000
```

如果能看到登录页，说明服务启动成功。

### 第二步：验证登录

使用以下账号登录：

```text
医生姓名：赵医生
密码：123456
```

登录后如果进入首页，说明登录功能正常。

### 第三步：验证发起提问

用 `赵医生` 登录后，进入“发起提问”页面。

填写：

```text
标题：测试会诊问题
病情描述：这是一个测试病例，用于验证 Demo 是否正常。
请教医院：任选一个医院
```

可选上传一张图片或一个 PDF。

点击“提交提问”。

如果跳转到“我的提问”，并且能看到刚提交的记录，说明提问功能正常。

### 第四步：验证附件

进入刚才提交的提问详情页。

如果上传的是图片或视频，应能看到预览。

如果上传的是 PDF 或其他支持文件，应能看到下载按钮。

### 第五步：验证回复

退出当前账号。

使用被请教医院的医生账号登录，例如：

```text
医生姓名：张主任
密码：123456
```

进入“会诊提问列表”，找到刚才的提问，点击查看详情。

填写回复内容并提交。

如果状态变为“已回复”，说明回复功能正常。

### 第六步：验证回复通知

退出当前账号。

重新登录原提问人：

```text
医生姓名：赵医生
密码：123456
```

如果页面顶部显示“您的提问已被回答”，说明通知功能正常。

### 第七步：验证权限

可以分别登录不同医生账号观察：

- 自己只能在“我的提问”中看到自己发起的提问。
- “会诊提问列表”只显示当前账号有权限查看的提问。
- 只有被指定为请教医院的医生可以回复。
